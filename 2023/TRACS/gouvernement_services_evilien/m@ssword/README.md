***** DESCRIPTION *****************************************************************************************************************************************

Pour neutraliser une flotte de drones du ministère de la défense d’EvilCountry, nous avons besoin de récupérer les mots de passe permettant de les reconfigurer à distance.
Nous avons récemment intercepté des échanges entre des ingénieurs militaires d’EvilCountry en charge de leur maintenance. Nous espérons que ces informations vous permettront de comprendre comment sont générés les mots de passe afin d’en retrouver un maximum.
***************************************************************************************************************************************************************************

 	Liste des mots de passe 		0 	0 / 1 	100 		Répondre
	

Nous attendons un texte dont le contenu ressemble à (avec beaucoup plus de drones dans votre cas et, on l’espère, les bons mots de passe) :

drone1 password1
drone2 password2
drone3 password3
drone4 password4
…

où le nom du drone et son mot de passe sont séparés par un espace et où les lignes sont séparées par des retours chariot.

Evidemment, plus vous trouverez de mots de passe, plus vous obtiendrez de points.

Si un nom de drone apparaît plus d’une fois, seule la première occurence sera évaluée.
